FAQ

Common questions

Does it really work without internet?
Yes — 100%. All data is stored in the browser's local storage using an offline-first architecture. The system runs identically whether or not you have an internet connection. Internet is only needed for the initial download.
How does LAN sync work? What do I need?
You need to install PocketBase (a free, ~30MB binary) on one computer in your hotel. NEXORA OS connects to it over your local WiFi or LAN. All hotel computers then share data in real time. We include a one-click setup script for Windows, Mac, and Linux.
Is there a room limit?
No. NEXORA OS supports unlimited rooms. You can configure any number of room types, floors, and price categories. The tape chart and room grid scale automatically.
Can I customise the invoice with my hotel's logo?
Yes. In Settings you can upload your hotel logo, set invoice header/footer text, VAT registration number, and choose currency. Invoices are generated as PDF and as a printable A4 page. An 80mm thermal receipt format is also available.
What languages does it support?
NEXORA OS includes a full Arabic (right-to-left) interface and English (left-to-right). You can switch languages from any screen. Invoices and reports are generated in whichever language is active.
Can multiple staff log in at the same time?
Yes. You can create unlimited staff accounts, each with their own login and permissions. The system supports admin and staff roles with 34 granular permission controls. Shift management tracks which employee is on duty.
What happens to my data if I reinstall the browser?
The system keeps automatic daily backups for 7 days in local storage. You can also export a full backup file at any time. When LAN sync is active, your data is also stored on the PocketBase server and can be fully restored.
Is this a subscription or a one-time purchase?
NEXORA OS is sold as a one-time licence. You own the software and can use it indefinitely. Lifetime updates are included in the Professional plan. The Starter plan includes 1 year of updates.
