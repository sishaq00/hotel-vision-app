# إضافة شبكة الغرف + لوحة الضيوف في الداشبورد

سأبقي تصميم النظام كما هو حالياً، وأضيف عنصرين جديدين أسفل بطاقات الـ KPI في صفحة الداشبورد (`src/routes/index.tsx`).

---

## 1) شبكة الغرف (Rooms Grid Panel)

مربع كبير يعرض **كل أرقام الغرف** بشكل أنيق ومرتب (أزرار مربعة صغيرة مع رقم الغرفة)، مع **مفتاح ألوان** يوضح الحالة:

- 🟢 **أخضر** — متاحة ونظيفة (يمكن إعطاؤها)
- 🟡 **أصفر** — متسخة (الضيف داخل أو عمل Check-out — لا يمكن إعطاؤها)
- 🔵 **أزرق** — مشغولة حالياً بضيف (in-house)
- 🔴 **أحمر** — Out of Order / صيانة

الترتيب حسب رقم الغرفة، مع تجميع اختياري حسب الطابق.

### عند الضغط على رقم غرفة:
يفتح Dialog يعرض حسب حالة الغرفة:

**أ) إذا كانت الغرفة متاحة (خضراء):**
- معلومات الغرفة: الرقم، النوع، السعر الافتراضي، الطابق، عدد الأسرّة
- زرّان رئيسيان:
  - **حجز/إعطاء الغرفة الآن** → يفتح `NewReservationDialog` مع تحديد الغرفة مسبقاً
  - **حجز مستقبلي** → نفس الحوار مع تواريخ مختلفة
- حقل تسعير مخصّص (custom rate) إذا أراد المستخدم تجاوز السعر الافتراضي

**ب) إذا كانت مشغولة (زرقاء):**
- معلومات الضيف الحالي + تواريخ الإقامة + إجمالي الفاتورة + المتبقي
- أزرار: عرض التفاصيل الكاملة / Check-out

**ج) إذا كانت متسخة أو OOO:**
- عرض الحالة فقط مع رسالة أنها غير متاحة للحجز

## 2) مستطيل عريض للضيوف (Today's Guests Strip)

أسفل شبكة الغرف — جدول مدمج بعرض كامل يعرض **كل الضيوف النازلين حالياً** بالأعمدة التالية:

| الضيف | رقم الغرفة | النوع | تاريخ الدخول | تاريخ الخروج | الليالي | الإجمالي | المتبقي | الحالة |

### الألوان:
- 🟢 **سطر بخلفية خضراء خفيفة** — الضيف باقٍ (Check-out بعد اليوم)
- 🔴 **سطر بخلفية حمراء خفيفة** — يخرج اليوم (Check-out = اليوم)

### عند الضغط على اسم الضيف:
يفتح Dialog شامل يعرض:
- معلومات الضيف الكاملة (الاسم، الهوية/الجواز، الجنسية، الهاتف)
- تفاصيل الحجز (الغرفة، التواريخ، عدد الليالي، السعر/ليلة)
- ملخّص الحساب (الإجمالي، المدفوع، المتبقي، الإيداعات)
- روابط سريعة: عرض الفاتورة، Check-out، تمديد الإقامة، صفحة الضيف الكاملة

## التفاصيل التقنية

**ملف واحد جديد فقط:**
- `src/components/dashboard/RoomsGridPanel.tsx` — شبكة الغرف + Dialog التفاصيل
- `src/components/dashboard/TodayGuestsPanel.tsx` — جدول الضيوف + Dialog معلومات الضيف

**تعديل واحد:**
- `src/routes/index.tsx` — إضافة المكوّنين بعد `DashboardCharts` وقبل `QuickAction`s

**مصادر البيانات** (كلها موجودة في `useHotelStore`):
- `rooms` — لتحديد ألوان الشبكة (status, housekeepingStatus, archived)
- `reservations` — للضيف الحالي في كل غرفة + لائحة الضيوف اليوم
- `guests` — أسماء الضيوف ومعلوماتهم
- `payments` (إن وُجد في الـ store) — لحساب المتبقي

**حوارات مُعاد استخدامها:**
- `NewReservationDialog` (مع prop `defaultRoomId`) للحجز السريع من الشبكة
- `CheckoutDialog` للـ checkout
- `ExtendStayDialog` للتمديد
- `Link to="/guest/$guestId"` للملف الكامل

**الاتساق البصري:**
استخدام نفس tokens الموجودة (`bg-success/10`, `bg-info/10`, `bg-warning/15`, `bg-destructive/10`, `Card`, `Dialog`, `Button`) — لا تغيير في الثيم العام.
